Week 5: Reflectie & Beslissingen

Doelen

Het doel van deze week was om mijn ontwerp uit Figma om te zetten naar HTML en CSS. Ik wilde de basisstructuur van de website opbouwen, werken met containers, en mijn gekozen kleuren en lettertypes uit Figma toepassen in de code. Daarnaast wilde ik leren hoe ik mijn project online kan zetten via GitHub en Netlify.

Wat heb ik gedaan?

  • De HTML-structuur van mijn project opgezet met verschillende pagina’s per week.
  • Een navigatiebalk toegevoegd zodat alle weken met elkaar verbonden zijn.
  • De lettertypes uit Figma (via Google Fonts) geïmporteerd in mijn CSS.
  • Een achtergrondkleur en basisopmaak toegevoegd in styles.css
  • >Mijn project gepubliceerd via GitHub en Netlify.

Redenen voor mijn keuzes

Ik heb ervoor gekozen om de opmaak rustig en overzichtelijk te houden, met voldoende witruimte tussen de tekst. Dat past bij de stijl die ik in Figma had uitgewerkt. De paarstint die ik eerder had gekozen heb ik ook in mijn CSS toegepast als accentkleur voor titels en knoppen. Door met HTML te werken, kreeg ik beter inzicht in hoe de structuur van een webpagina is opgebouwd, en hoe de CSS die vormgeving bepaalt.

Lessen & Next steps

Ik heb geleerd hoe belangrijk het is om eerst een duidelijke structuur te hebben voor je HTML-bestanden. Ook begrijp ik nu beter hoe ik lettertypes en kleuren vanuit Figma kan vertalen naar code. Volgende week wil ik meer experimenteren met layout en spacing in CSS, zodat de website nog professioneler oogt.
