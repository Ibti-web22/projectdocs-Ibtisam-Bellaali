Welkom

Welkom op mijn projectdocumentatie voor het vak Lost in Tra(i)nslation. Op deze website houd ik mijn proces en evolutie per week bij. Elke week beschrijf ik wat ik gedaan heb, welke keuzes ik maakte en wat ik daaruit heb geleerd.

Het project draait om het visueel vertalen van een thema of idee op een creatieve manier. In dit vak leer ik stap voor stap hoe een concept zich ontwikkelt van ruwe schets tot digitaal ontwerp, en uiteindelijk naar een uitgewerkte website met HTML en CSS.

Klik links in het kader op een week om mijn voortgang te volgen. Je ziet hoe mijn ideeën groeiden, welke feedback ik kreeg en hoe ik mijn ontwerp telkens heb verbeterd.

Structuur

  • Elke week heeft een eigen HTML-pagina met reflectie en beslissingen.
  • De navigatie staat links zodat je makkelijk kunt doorklikken.
  • Het project wordt stap voor stap gepubliceerd via GitHub en Netlify.
  • De website is eenvoudig opgebouwd: de focus ligt op inhoud en structuur.

Over mezelf

Ik ben Ibtisam Bellaali, student grafische vormgeving. In dit vak leer ik niet alleen over ontwerpen, maar ook over hoe je je werk logisch opbouwt, presenteert en technisch vertaalt naar een webomgeving. Ik vind het interessant om te zien hoe elk detail — van kleur tot typografie — invloed heeft op de sfeer en leesbaarheid van een ontwerp.

Mijn doel is om de komende weken mijn ontwerp verder te verfijnen, mijn HTML-structuur beter te begrijpen en te leren hoe ik met Tailwind CSS de juiste balans kan vinden tussen vorm en inhoud.
